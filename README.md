# yktool

## jdklink

### 一键获取opensdk 下载地址

```shell
yktool jdklink -o mac -x aarch64 -v 20  
```

## ghcdn

### github 文件加速

```shell
yktool ghcdn -u address
```

## finacne

### 获取24小时内货币汇率

```shell
yktool finance -s USD -t CNY -n [数量] -f [过滤]
```

## aes

### AES加解密文件

```shell
yktool aes -f [加解密文件] -p [密钥] -m [加解密模式]
```
